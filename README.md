# 🚀 AI Adoption Analytics Platform

**Data Mining Course Project** - RFM Analysis, K-Means/Hierarchical Clustering, Association Rules

## 📋 Features
- **File Upload** → Live RFM computation
- **Interactive Visuals**: 3D RFM Scatter, Dendrogram, Sankey, Heatmap
- **Client Personas** + What-If Revenue Simulator
- **Export** prioritized outreach CSV

## 🚀 Quick Start

```bash
# Clone & install
git clone <repo>
cd ai-adoption-analytics
pip install -r requirements.txt

# Run locally
streamlit run app.py
